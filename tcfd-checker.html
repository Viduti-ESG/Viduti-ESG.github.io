⚡ Backend Not Connected
AI analysis requires the Green Curve backend. Start it with:
ANTHROPIC_API_KEY=sk-... uvicorn ai_api:app --reload --port 8002

Then set the API base URL in the dashboard Settings, or open the console and run:
localStorage.setItem('gc_api_base', 'http://localhost:8002')

Without the backend, this tool shows a manual TCFD checklist instead.
